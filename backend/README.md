# Bla Bla Ride Sharing App

The **Bla Bla Ride Sharing App** is a university project developed at **HNU** for the **Enterprise Application Engineering Project** course. The application follows a clientâ€“server architecture where a mobile application communicates with a backend service to support ride-sharing functionality.

## Project Details
- **Project Name:** Bla Bla Ride Sharing App
- **University:** HNU
- **Course Name:** Enterprise Application Engineering Project

## Technology Stack
- **Frontend:** React Native
- **Backend:** Spring Boot
- **Programming Language:** Java
- **Build Tool:** Maven

## Prerequisites
Before running the project, ensure the following are installed:
- Java (JDK 17 or later recommended)
- Maven

how to isntall maven
- sudo apt install maven (ubuntu)
 - choco install maven (windows)

## Check installation:

java -version
mvn -version

## Run the server run:

mvn spring-boot:run

The Spring Boot application will start and expose backend APIs that are consumed by the React Native frontend.
